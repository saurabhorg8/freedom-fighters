# Freedom Fighters
Web page displaying freedom fighters of India

## The freedom fighters added in the project:
- Mahatma Gandhi
- Subhas Chandra Bose
- Bhagat Singh
- Vallabhbhai Patel
- Chandrashekhar Azad
- Lal Bahadur Shastri

## Contribution guide
Add the freedom fighter in the given format of card
- Square Picture
- Name
- Description
- Wikipedia link
